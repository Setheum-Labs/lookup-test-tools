## Installation

```shell
npm install --save-dev ethcore/test-tools
```

## Usage

You need a running & synced [Parity](https://ethcore.io/parity.html) at `localhost:8545`. Run with ` --jsonrpc-apis eth,parity` to enable the necessary RPCs.
